# AI视频导演必看书籍（零基础→专业，直接照着买）
## 一、入门基础（必看，懂镜头语言）
1. **《认识电影》**
    影视入门天花板，镜头、构图、剪辑、场面调度全讲透，AI拍视频最缺**镜头思维**
2. **《镜头语言：从入门到精通》**
    通俗好懂，景别、运镜、光影、视角，直接套用AI提示词
3. **《电影语言的语法》**
    导演核心工具书，分镜、机位、画面组合，做AI短片必备

## 二、导演核心能力（叙事+分镜）
1. **《导演工作》**
    实战导演思维，剧本拆解、场面调度、情绪把控
2. **《分镜头脚本设计教程》**
    重中之重！AI视频**先写分镜再生成**，这本书直接教画分镜、写镜头脚本
3. **《故事》**
    罗伯特麦基，做短视频/短剧AI视频必学，学会讲故事才有流量

## 三、剪辑&视听节奏（AI成片关键）
1. **《看不见的剪辑》**
    把控视频节奏、转场、情绪卡点，AI生片最容易节奏乱
2. **《剪辑的语法》**
    简单实用，快速掌握剪辑逻辑

## 四、AI视频专属进阶（现在最实用）
1. **《AI短视频创作实战》**
    贴合现在Sora、Runway、即梦、剪映AI，提示词+镜头搭配实操
2. **《数字影像创作》**
    光影、色调、画质调校，提升AI视频电影质感

## 五、短剧/剧情向AI导演首选
- **《短剧剧本创作与拍摄》**
  现在AI流量主力是短剧，学会写短剧本、控时长、抓冲突

# 最简学习路线
1. 先读《认识电影》建立画面感
2. 再学《分镜头脚本》定镜头
3. 看《故事》写剧情
4. 用剪辑书籍调成片节奏
5. 最后结合AI创作书写专业镜头提示词
