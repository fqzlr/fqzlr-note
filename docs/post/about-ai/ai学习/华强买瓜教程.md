
# AI 令狐买瓜视频全流程制作笔记
**视频来源**：B 站 BV1HoLG62ET1 **UP 主**：索尔 FM **核心内容**：武侠风 AI 买瓜二创完整制作步骤 + 踩坑避坑 + 新手学习路线
## 一、创作整体思路
1. 初衷：想用 AI 制作热门买瓜二创视频，选择武侠改编路线，属于高难度开局
2. 定位：融合《笑傲江湖》武侠风格 + 经典买瓜剧情，兼具趣味与风格化  
3. 核心总结：AI 降低单人成片门槛，但影视专业知识、镜头思维依旧是作品质量核心

## 二、六大制作步骤
### 1. 剧本改编
- 参考热门买瓜视频：搭配洗脑顺口歌曲，节奏轻快搞笑
- 选材：以《笑傲江湖》为剧本蓝本进行二次改编
- 踩坑点：选用央视版主题曲风格深沉，不够魔性顺口，偏自嗨，适配搞笑二创效果一般
### 2. 角色选角设计
- 混搭多版本经典武侠角色
    - TVB 版：令狐冲、向问天
    - 央视版：任盈盈、任我行
    - 影视版：蓝凤凰、东方不败
- 制作思路：保留原版人物服饰特征，直接和买瓜剧情人物形象融合

![image.png](https://tu.fqzlr.com/file/blog/wenzhang/1779115046570_image.png)
### 3. 场景搭建
- 取景原型：央视版笑傲江湖令狐冲与任盈盈初见乡村场景
- 统一风格：沿用原版影视色调画风
- 技术手段：**AI 扩图 + 九宫格视图校准**，保证全片场景画风一致
    ![image.png](https://tu.fqzlr.com/file/blog/wenzhang/1779201746732_image.png)
### 4. 音频人声改编（歌曲换词核心）

1. 工具：**MSST1.7** 多层音轨分离
    
    1. 第一层：人声 + 背景音分离
        
    2. 第二层：主唱 + 和声分离
        
    3. 第三层：主唱干声 + 和弦分离
        
2. 后期处理：提纯干净人声干声
    
3. 音色复刻改词：使用 **AC Studio** 做 AI 声音模拟，完成歌词替换改编
    
4. 难点：原版原唱声线深邃，大幅度改词调音难度极高
    

### 5. AI 视频生成（核心实操）

#### 所用平台 & 模型

- 平台：B 站 UpDream（即梦）
    
- 生图模型：大香蕉 2.0
    
- 动态视频模型：CDance2.0
#### 分镜制作经
- 武打 MV 类：无需提前画分镜，文字描述清晰即可，AI 易出创意效果
- 对话剧情类（买瓜视频）：**必须提前做好分镜**
    - 优势：节省平台 Token 积分、人物动作台词更贴合
    - 提示词要点：写明景别、运镜方式；追求个人风格可标注导演常用摄影机 + 镜头型号

#### 高频避坑要点
1. 多人对话镜头极易出现**音画台词错位**
2. 无分镜图制作，容易出现人物站位、动作不符合剧情逻辑
3. 非必要禁用「全能参考」：扣除积分，还会持续拉低后续画面质量
4. AI 幻觉问题：文案出现经典广告语，容易导致人物面部形象错乱
5. 成片通病：AI 生成素材可用片段短，大多仅中间几秒可用
### 6. 后期剪辑收尾
1. 素材处理：掐头去尾筛选可用片段，全片节奏依靠后期剪辑把控
2. 学习参考
    1. 武侠节奏：拉片徐克作品
    2. 快嘴对话、强张力剧情：学习姜文、昆汀影片剪辑逻辑    
3. 核心：剪辑审美与镜头语感只能靠长期拉片 + 实战积累

## 三、新手学习进阶路线（UP 主亲授
1. 入门工具：先学习本地部署 AI 生图 / 生视频工具，优先学 ComfyUI
2. 模仿练习：选定喜欢的影视片段，拆解剧情、人物、景别、运镜、配乐
3. 日常积累：观影时刻意分析光影色调、人物站位、镜头情绪搭配
4. 低成本起步：先免费工具练手，熟练后再开通平台会员付费制作
